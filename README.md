# Tecem - Dashboard Frontend 

Interfaz de usuario moderna para el sistema **Tecem**, optimizada para el llenado de calificaciones y gestión académica. Construida con **React** y el sistema de diseño **MUI (Material UI)**.

## ✨ Características de la Interfaz
- **Panel de Control:** Visualización de métricas de rendimiento escolar.
- **Formularios Dinámicos:** Llenado ágil de actas de calificaciones.
- **Estado Global:** Gestión de datos con Redux y persistencia de sesión.

---

## 🛠️ Instalación y Configuración

Asegúrate de tener el backend (`tecem-back`) en ejecución para que los datos se visualicen correctamente.

### 1. Instalar dependencias
Entra al directorio y usa tu gestor de paquetes preferido:

$ cd tecem-front

$ npm install
# o
$ yarn install

### 2. Iniciar en modo desarrollo
$ npm start
# o
$ yarn start

La aplicación se abrirá automáticamente en: http://localhost:3000
